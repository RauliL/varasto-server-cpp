# Varasto server C++ implementation

C++17 implementation of [Varasto server]. Work in progress.

[Varasto server]: https://www.npmjs.com/package/@varasto/server

## Compilation

```bash
$ git submodule update --init
$ mkdir build
$ cd build
$ cmake ..
$ make
```
